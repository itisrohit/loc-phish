## Setup & Development

### 1. Copy Environment Variables
```bash
cp .env.example .env
```

### 2. Local Development
```bash
npm install
npm run dev
```

### 3. Production Build
```bash
npm run build
```

---

## IP Geolocation Lookup Example
To lookup geolocation details for a public IP address, you can run:

```bash
curl http://ip-api.com/json/<IP_ADDRESS>
```
